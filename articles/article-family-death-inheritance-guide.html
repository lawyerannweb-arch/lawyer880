📝 家人過世後的立即行動

緊急時間表：不能錯過的關鍵期限

  • 24小時內：取得死亡證明書（醫院或警察機關）
  • 30天內：辦理死亡登記（戶政事務所）
  • 3個月內：決定是否拋棄繼承（法院）
  • 6個月內：申報遺產稅（國稅局）
  • 1年內：完成遺產分割協議

第一步：取得死亡證明書

🏥 自然死亡（醫院）

  • • 由主治醫師開立死亡證明書
  • • 通常在醫院當天就能取得
  • • 需要家屬身分證明文件

🚨 意外死亡（檢察官）

  • • 需要檢察官相驗
  • • 時間較長，可能需數天
  • • 由檢察官開立相驗屍體證明書

第二步：辦理死亡登記

💼 需要準備的文件

  • ✓ 死亡證明書正本
  • ✓ 戶口名簿
  • ✓ 申請人身分證
  • ✓ 印章
  • ✓ 死亡登記申請書
  • ✓ 關係證明文件

小提醒：死亡登記完成後，記得申請「除戶戶籍謄本」，後續的繼承手續都會用到。建議一次申請10-15份，因為銀行、地政事務所等各機關都需要正本。

步驟一：確認繼承人與繼承權

首先要搞清楚「誰有繼承權」。很多人以為只有兒子能繼承，這是錯誤的觀念！台灣的繼承制度是男女平等的。

步驟二：遺產清查

繼承可不只是分財產這麼簡單，債務也會一起繼承！所以一定要先搞清楚到底有多少財產和債務。

💰 積極財產（資產）

  • • 銀行存款、定存
  • • 不動產（房屋、土地）
  • • 股票、基金
  • • 保險理賠金
  • • 債權（別人欠的錢）
  • • 汽車、珠寶等動產

💸 消極財產（債務）

  • • 銀行貸款
  • • 信用卡債務
  • • 房屋貸款
  • • 保證債務
  • • 未繳稅款
  • • 其他借款

實用小技巧：可以向國稅局申請「被繼承人財產清單」，裡面會列出大部分的財產資料，但不包括現金和債務，這些需要自己調查。

💰 遺產稅計算與申報

很多人聽到「遺產稅」就頭痛，其實只要掌握公式，計算並不難。而且大部分家庭都不用繳遺產稅！

🧮 遺產稅計算公式

應納遺產稅額 = (遺產總額 - 免稅額 - 扣除額) × 稅率

2025年最新免稅額度

項目 金額 說明
免稅額 1,333萬元 基本免稅額度
配偶扣除額 493萬元 有配偶可扣除
直系血親卑親屬扣除額 每人 49.3萬元 子女、孫子女等
父母扣除額 每人 123萬元 被繼承人父母

遺產稅率表

遺產淨額 稅率 累進差額
5,000萬元以下 10% 0
超過5,000萬元部分 15% 250萬元
超過1億元部分 20% 750萬元

💡 實際案例計算

案例：張先生過世，留下3,000萬元遺產，有配偶和2個子女

計算步驟：

  • 遺產總額：3,000萬元
  • 免稅額：1,333萬元
  • 配偶扣除額：493萬元
  • 子女扣除額：49.3萬 × 2人 = 98.6萬元
  • 總扣除額：1,333 + 493 + 98.6 = 1,924.6萬元
  • 遺產淨額：3,000 - 1,924.6 = 1,075.4萬元
  • 應納稅額：1,075.4萬 × 10% = 107.54萬元

🛡️ 限定繼承 vs 拋棄繼承

這是最多人搞不懂的部分！簡單來說：限定繼承就像是「有條件的繼承」，而拋棄繼承就是「完全不要」。

限定繼承

  • 📋 定義：只在遺產範圍內負擔債務
  • 💰 好處：可以繼承財產，債務不會超過遺產
  • ⏰ 期限：自動適用，無需申請
  • 📝 手續：2009年後為法定繼承方式
  • 🎯 適合：遺產大於債務的情況

拋棄繼承

  • 📋 定義：完全放棄繼承權
  • 💰 結果：財產和債務都不繼承
  • ⏰ 期限：3個月內向法院聲請
  • 📝 手續：需要向法院提出申請
  • 🎯 適合：債務遠大於遺產的情況

2

重要警告：拋棄繼承有不可逆性！一旦向法院聲請通過，就無法撤回。而且拋棄繼承後，你的子女也會跟著失去代位繼承的權利。

👥 繼承權順序完整說明

很多家庭因為不了解繼承順序而產生紛爭。其實法律規定得很清楚，讓我們用圖表來說明：

🏆 繼承權順序排行榜

👑

配偶：永遠的冠軍

無論什麼情況，配偶都有繼承權，只是比例會隨著其他繼承人而調整。

1

第一順位：子女及其後代

  • • 親生子女、養子女、非婚生子女（一視同仁）
  • • 子女已死亡時，由孫子女代位繼承
  • • 配偶得1/2，子女平分另外1/2
2

第二順位：父母

  • • 只有在沒有第一順位繼承人時才輪到
  • • 配偶得1/2，父母平分另外1/2
  • • 包括養父母
3

第三順位：兄弟姊妹

  • • 同父同母、同父異母、同母異父都算
  • • 配偶得1/2，兄弟姊妹平分另外1/2
  • • 兄弟姊妹已死亡時，由其子女代位繼承
4

第四順位：祖父母

  • • 最後順位的繼承人
  • • 配偶得2/3，祖父母平分另外1/3
  • • 包括外祖父母

💡 實際案例分析

案例一：王先生過世，有配偶、2個子女、父母健在

→ 繼承人：配偶 + 2個子女（父母無繼承權）

→ 分配：配偶1/2，每個子女各1/4

案例二：李小姐過世，未婚無子女，父母已故，有3個兄弟姊妹

→ 繼承人：3個兄弟姊妹

→ 分配：每人各1/3

❓ 常見問題FAQ

Q1：限定繼承還需要辦理嗎？

A：不需要！自2009年5月22日起，台灣已經改為「當然限定繼承」制度，也就是說，所有繼承都自動適用限定繼承，繼承人只在遺產範圍內負擔債務責任，不會讓個人財產受到影響。

Q2：如果3個月內沒有拋棄繼承會怎樣？

A：超過3個月期限就無法拋棄繼承了，會被視為接受繼承。不過由於現在是限定繼承制度，所以不用太擔心，債務責任不會超過遺產範圍。除非債務遠大於遺產，否則建議諮詢律師評估。

Q3：遺產稅一定要繳嗎？

A：不一定！2025年遺產稅免稅額是1,333萬元，加上各種扣除額（配偶493萬、每個子女49.3萬等），大部分家庭的遺產都在免稅範圍內。但即使不用繳稅，還是要在6個月內申報。

Q4：女兒真的和兒子有一樣的繼承權嗎？

A：是的！台灣法律明確規定男女繼承權平等，不管是兒子還是女兒，繼承權完全相同。傳統「兒子才能繼承」的觀念是錯誤的，已經不符合現行法律。

Q5：配偶和子女誰的繼承權比較大？

A：配偶和子女都是第一順位繼承人，會一起繼承。分配比例是：配偶得1/2，所有子女平分另外1/2。例如有配偶和2個子女，配偶得1/2，每個子女各得1/4。

Q6：如果繼承人之間意見不合怎麼辦？

A：首先嘗試協調談判，可以請律師協助調解。如果真的無法達成協議，可以向法院聲請「遺產分割」，由法官依法分配。不過訴訟曠日費時，建議盡量以協商方式解決。

Q7：可以只繼承財產不繼承債務嗎？

A：不可以單獨選擇！繼承是「概括繼承」，財產和債務必須一起承受。但在限定繼承制度下，債務責任不會超過遺產範圍。如果債務太多，可以選擇拋棄繼承。

Q8：銀行帳戶什麼時候會被凍結？

A：當銀行得知帳戶持有人死亡消息時，就會凍結帳戶。家屬需要提供除戶戶籍謄本、繼承系統表、遺產分割協議書等文件才能解凍。建議盡早處理相關手續。
